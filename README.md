# 📱 Unicode → GSM Converter

A modern, responsive web tool to convert Unicode text into **GSM-7 encoding**, detect unsupported characters, and accurately calculate SMS length and parts.

Perfect for developers, marketers, and businesses sending SMS who want to **optimize message cost and compatibility**.

---

## ✨ Features

- 🔤 **Unicode → GSM-7 Conversion**
  - Converts unsupported characters into GSM-compatible equivalents

- ⚠️ **Non-GSM Character Detection**
  - Highlights characters that break GSM encoding (like emojis, smart quotes)

- 📊 **SMS Length & Parts Calculator**
  - Calculates:
    - Character count
    - Encoding type (GSM-7 / Unicode)
    - Number of SMS parts

- 🔄 **3 Conversion Modes**
  - **Replace** → Converts unsupported characters
  - **Strip** → Removes unsupported characters
  - **Keep** → Keeps text but highlights issues

- 📉 **Cost Comparison Panel**
  - Compare original vs converted SMS length and parts

- 🎯 **Smart Replacement Engine**
  - Automatically replaces:
    - “ ” → "
    - ‘ ’ → '
    - — → -
    - … → ...
    - Accented characters → GSM equivalents

- 🌗 **Dark / Light Mode Toggle**

- 📋 **Copy Output Button**
